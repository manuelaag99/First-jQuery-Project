jQuery: "after", "before", "append", and "prepend"

As you know, when using jQuery, you can add code around existing code, by using "after", "before", "append", and "prepend", but some forget what each one actually does. This website helps to illustrate it, in case you need to remember. Let's suppose you want to add a "button" tag with the text "Click here!". The left side shows what your HTML code is transformed into, while the right side shows how that code will be represented on your website.
